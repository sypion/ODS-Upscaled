# ODS-Upscaled
A content pack made to upscale every texture for Operation Dessert Storm.
This Git repository is made for reference only during this time. If you'd like to download the content pack, [click here.](https://mega.nz/#!hQ4A0LII!sf2XHNM35GBnyK7iGhSECIsD0wNBPhcmLBfaMLyfFQw)

#### Take Notice:
This is the first version of the content pack, and you might run into multiple issues. It is highly recommended you [visit the Issues portion of the repository](https://github.com/sypion/ODS-Upscaled/issues) if you encounter something wrong. This will help the contributors find and solve the problem as soon as possible.

## How to install:

Windows:

```
1. Visit C:/Program Files (x86)/Operation Dessert Storm/resources
2. Create a folder inside the resources folder called "Upscaled"
3. Put all the .mf files into the "Upscaled" folder
4. Run the game and have fun!
```
